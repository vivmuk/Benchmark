Methodology

How we score

Every benchmark is run with fixed prompts, default sampling, and transparent heuristics. Scores are reproducible and auditable.

1 · Intent Understanding

Models receive ambiguous, underspecified prompts. Points for clarifying questions, structured proposals, and explicit uncertainty before assuming details.

2 · One-Shot UI Generation

Models must emit a single self-contained HTML file. Scored on visual polish, semantic structure, accessibility, responsiveness, and whether it runs without edits.

3 · Long-Horizon Agentic Task

Models plan and outline a multi-step tool-calling system. Points for step coverage, tool mentions, failure-mode awareness, and logical ordering.

4 · Brick Breaker Realism

Models generate a playable Canvas game in one shot. Scored on physics plausibility, control responsiveness, collision detection, game states, particle/sound effects, and responsive design.
